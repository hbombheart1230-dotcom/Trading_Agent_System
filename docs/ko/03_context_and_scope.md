# 3. 범위 / 비범위 / 가정

## 3.1 범위(In Scope)
- 키움 REST 기반 mock/real 라우팅
- 승인(수동/자동) 및 가드 체계
- 의도(OrderIntent) 중심 실행 모델
- 이벤트 로그 기반 관측/리포팅
- M1~M15 단계 산출물 통합

## 3.2 비범위(Out of Scope)
- 고빈도/초저지연(HFT) 최적화
- 다중 브로커(키움 외) 완전 지원 (향후 확장 가능)
- 실시간 뉴스 크롤링/추론 품질 최적화(Agent 고도화 영역)

## 3.3 운영 가정(Assumptions)
- 운영자는 env/설정 파일을 관리할 수 있다.
- 시크릿은 절대 Git에 커밋되지 않는다.
- mock 환경에서 충분히 검증 후 real에 진입한다.
